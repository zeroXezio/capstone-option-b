# Futuro Digital LatAm — Capstone Project Financial Wellness

## Goal
Transform raw financial survey data from 500 young professionals across Latin America into a comprehensive executive analysis report to shape future financial literacy programs.

## Environment & Tech Stack
- Python Version: 3.14.6
- Core Libraries: pandas, matplotlib, seaborn, scipy

## Naming Conventions
- Scripts: `scripts/01_explore.py`, `scripts/02_clean.py`, `scripts/03_analyse.py`
- Charts: `charts/01_income_by_country.png`, `charts/02_age_vs_savings.png`

## Data Files
- Raw File: `data/latam_finanzas_2025.csv`
- Clean File: `data/latam_finanzas_clean.csv`
